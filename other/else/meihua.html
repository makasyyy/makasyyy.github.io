梅花与六爻的区别 #

  1. 它两共用 64 卦的卦象，解读方法不同
  2. 梅花易数起卦，不配六亲、六神、它是以上下卦理（体用）、万物类象、卦辞来推断吉凶祸福
  • 重点：
    • 需要看卦辞
    • 看细节需要结合体用，体用关系贯穿始终
    • 代表著作：《梅花易数》
  1. 六爻卦-重在六个爻，必须配六亲、六神，不以卦象、卦辞论断，是以六个爻的动变，结合日月来论吉凶祸福
  • 重点：
    • 不需要看卦词
    • 看六神、六亲、应世爻，六个爻的变化
    • 代表著作：《增删卜易》《卜筮正宗》《黄金策》等

理论基础 #

太极生两仪、两仪生四象、四象生八卦、八卦定吉凶、吉凶生大业

  1. 太极-混沌
  2. 两仪-天和地，阴与阳，阳爻 —— 代表天 ，阴爻 -- 代表地
  3. 四象-代表四时，即春夏秋冬，具体到六爻上，就是少阴，少阳，老阴，老阳
  4. 八卦-在四象的基础上，再重上一个阴爻或阳爻，就形成了八个符号 即：乾、坎、艮、震、巽、离、坤、兑，他们分别代表自然界的八种物质： 乾为天、坎为水、艮为山、震为雷、巽为风、离为火、坤为地、兑为泽
  5. 八卦定吉凶。
  6. 何为六爻 符号为 ——阳爻 和--阴爻，爻可以理解为交，代表世间万物在不断地交流变化，是八卦地基本单位，由于八卦只能代表 8 种自然现象，所以需要将两个卦结合起来，称为复卦，一共有八八六十四种复卦，复卦中地两卦按照一上一下地方式组合，复卦共有六个爻位，因此又称六爻
  7. 六爻和梅花都是从底部往上排：初爻、二三四五六爻

先天八卦&后天八卦 #

  1. 先天八卦传说是伏羲画出来地，所以称为伏羲八卦，后天八卦起源于商朝时期地周文王，后天八卦是由先天八卦演变而来地，称为文王卦
  2. 先天八卦是空间地物象，天地定位，乾卦为天，天在上，坤卦为地，地在下
  3. 而后天八卦属于循环过程，它体现了四季地变化和万物生长，后天八卦与古时地农业生产有着密切地关系，先天八卦图表现为阴阳对峙关系，后天八卦图表现为五行相生，变易发展之道 ，总之，先天为体，后天为用，所以：我们需要记忆理解运用地，是后天八卦

    卦名：‘乾、坤、坎、离、震、巽、艮、兑’

    对应：‘天、地、水、火、雷、风、山、泽’

八卦的五行&代表意义 #

"天地定位，山泽通气，雷风相薄，水火不相射。" ————《易经》

  1. 天地定位
    • 乾，三爻全阳，表示全动，是天体之象
    • 坤，三爻全阴，表示全静，是大地之象
    • 先民的宇宙观是“天动而地静”，天在上而地在下，即“天地定位”
  2. 山泽通气
    • 艮 上爻为阳爻，中下爻为阴爻。上面小部分动，下面大部分动，为“山”之象
    • 兑 上爻为阴爻，中下爻为阳爻，上面小部分静，下面大部分动，为“降雨"之象
    • 山气上腾，雨水下降，即”山泽通气“。泽即是雨，雨降于天，山出于地，因此兑紧接乾后面，艮紧邻坤
  3. 雷风相薄
    • 震 下爻为阳爻，中上爻为阴爻。上面大部分静，下面少部分动，为“雷”之象。古人认为雷从地中起，所以震排在乾的旁边
    • 巽 下爻为阴爻，中上爻为阳爻。上面大部分动，下面少部分静，为“风”之象，古人认为风起天上，所以巽也排在乾的旁边
  4. 水火不相射
    • 坎 中爻为阳爻，上下爻为阴爻。周边静而地中动，为“水”之象
    • 离 中爻为阴爻，上下爻为阳爻。周边动而体中静，为“火”之象，又为“日”之象

      以上，都是先天八卦，作了解即可。下面即将解释的后天八卦，需要着重理解，我们几乎所有的运用，都是来自后天八卦的意象。

在后天八卦中，乾坤为父母，其余六卦皆为其子女。 乾为天，坤为地。

  1. 乾卦，天为父，代表爸爸，五行属金
  2. 坤卦，地为母，代表妈妈，五行属土
  3. 震卦，长男（大儿子），五行属木
  4. 坎卦，中男（二儿子），五行属水
  5. 艮卦，少男（小儿子），五行属土；地母坤也是土，坤妈最爱小儿子
  6. 巽卦，长女（大女儿），五行属木（长子长女，五行都是木）
  7. 离卦，中女（二女儿），五行属火
  8. 兑卦，少女（小女儿），五行属金；天父乾也是金，乾爸最爱小女儿

  • 后天八卦的代表意义 #

  1. 乾卦 乾象征的是时刻在运动的宇宙，也是万物万象焕发生机的原动力，有健康之意
    1. 就人体而言，乾卦代表人的头脑、中枢神经、思维，象征头、骨，是以人体之主精神能源

    3. 就季节而言，代表秋冬相交
    4. 就颜色而言，是大红、金黄
    5. 就人物而言，代表君父、大人、长者、领导等
    6. 就地理而言，后天方位-西北方
  2. 坤卦，坤卦象征地，阴柔、静止、顺从天的运转，为顺从之意
    1. 就人体而言，坤代表腹部
    2. 就五行而言，代表着土
    3. 就季节而言，代表着夏秋之交
    4. 就颜色而言，表示黄色
    5. 就人物而言，代表母亲、臣民等
    6. 就地理而言，后天方位-西南方

    3. 就季节而言，代表春天
    4. 就颜色而言，代表青色、绿色

  4. 巽卦，象征风，有渗透、无孔不入之意

    4. 就颜色而言，代表蓝色

    1. 就人体而言，代表耳朵

  8. 兑卦 ，是泽，为有水的沼泽地，这里是各种生物聚会的场所，能养鱼，且能接纳各种生命、有愉悦之意
    1. 就人体而言，象征口、舌、肺、痰涎

梅花卦具体操作步骤 #

  • 孔令伟梅花易数
  • 年月日时起卦
    • 上卦- 农历年月日总和除以 8，以余数为卦数求上卦
    • 下卦- 年月日时总和除以 8，余数为卦数求下卦
    • 动爻- 年月日时总和除以 6，余数为动爻
    • 八卦对应的数值：乾-1、兑-2、离-3、震-4、巽-5、坎-6、艮-7、坤-0
    • 十二地支序列： 子丑寅卯辰巳午未申酉戌亥
    • eg:阳历：2022，8.4，20：00
      • 即阴历：壬寅年，七月初七，戌时
      • 上卦：寅年取 3，戌时取 11 （年+月+日）÷ 8 余数为 1，为乾卦，则上卦为乾
      • 下卦：余数为 4，震卦，上下卦结合形成本卦
      • 动爻：（年+月+日+时）÷6 余数 4，动爻是为了看变卦的，改变其中一爻，由本卦得到变卦；4 为第四爻的含义。动爻在第四爻，从下往上数，第四爻由阳爻变为阴爻
      • 那互卦怎么来的呢? 互卦由主卦的二、三、四爻组成下卦即艮卦，主卦的三四五爻组成上卦即巽卦，上巽下艮，山风渐
  • 报数起卦，一般起两个 50 以内的数字
  • 解卦：
    • 排在第一个的是【本卦】，代表当下境遇
    • 排在第二个的是【互卦】，表示事情发展的过程
    • 最后是【变卦】，代表事情的结果
  • 预测结果最终以变卦为准，此为结果
    • eg：占得 本卦-天雷无妄，互卦-凤山渐，变卦-风雷益
  • 卦的应期一般都是近期的事情 3 个月左右
  • 注意事项：
    • 不动不占；不响不占；不问不占；
    • 不因事不占；心念未动不占
    • 总之，没事不要占

梅花占失物（上） #

《梅花易数》 卷二之失物占

  • 占失物，以体为主，用为失物。体克用，可寻，迟得；用克体，不可寻。体生用，物难见；用生体，物易寻。体用比和，物不失矣。
  • 又以变卦为失物之所在。
  • 如变是乾，则觅于西北，或公廨楼阁之所。或金石之傍，或圆器之中，或高亢之地。
  • 变是坤，则觅于西南方，或田野之所。或仓廪之处。或稼墙之处，或土窖穴藏之所。或瓦器方器之中
  • 震则寻于东方。或山林之所，或从棘之内，钟鼓之傍。或闹市之地。或大途之所。
  • 巽则寻于东南方，或山林之所，或寺观之地。或菜蔬之园，或舟居之间。或木器之内。
  • 坎则寻于北方。多藏于水边或溪井沟渠之处。或酒醋之边。或鱼盐之地。
  • 离则寻于南方。或庖厨之间，或炉冶之傍。或在明窗。或遗虚室。或在文书之侧。或在烟火之地
  • 艮则寻于东北方。或山林之内。或近路边。或岩石傍，或藏土穴。
  • 兑则寻于西方。或居泽畔，或败垣破壁之内。或废井沼泽之中。

梅花占失物（中） #

在梅花易数中，体用之间的关系是直接影响卦象判定的基本原则，体和用的明确和生克关系始终贯穿着整个占卜体系，因此能够明确体用，明辨生克才可断吉凶

那么何谓体？何谓用？

体就是自己本体，而用则是我们爻占卜的对象事物

比如我要占卜是否可以有财运，体就是我自己，用就代表了财运。

那么体何用如何分呢？

在一个卦象中，主卦是由六个爻组成的，这六个爻中某个爻变动了，就产生了体和用

【用卦】主卦中发动之爻所在的卦

【体卦】主卦中不发动的卦

诀窍：

第一爻到第三爻有变爻的就属于下卦为用，上卦为体。

从第四爻到第六爻有变的就属于上卦为用，下卦为体。

简单的说，一个卦的上卦和下卦中，爻变在上卦的范围内，则上卦是用，反之则下卦是用。

总之，不变的为【体】，变化的是【用】

  • 体用关系，五大规则：
    1. 体克用，事情吉。就如同我打赢了人家
    2. 用克体，事情凶，人家打赢了我
    3. 体生用，事情凶，就如同我把钱给人家
    4. 用生体，事情吉，人家把钱给了我
    5. 体用比合，以吉算，就如同大家相敬如宾
  • 以上规则用在寻找失物中：（占测失物能否找到，体卦为自己，用卦代表失物）
    1. 体克用，失物可以找到，但需要一点时间
    2. 用克体，失物不能找到
    3. 体生用，失物难以找到
    4. 用生体，失物容易找到
    5. 体卦用卦比和，物品并没有丢失
  • 以上的生、克、比和，指的是卦五行的关系
    • 克：指五行相克，金木水火土，金克木，木克土，土克水，水克火，火克金
    • 生：指五行相生，金水木火土，金生水，水生木，木生火，火生土，土生金
    • 体用比和：指五行相同：比和卦，指体卦和用卦为同一种五行重叠的卦，即上卦和下卦的五行相同 练习时间：

得卦： 本卦-雷山小过 互卦 -泽风大过 变卦-雷火丰

本卦：雷山小过，上卦雷卦，下卦艮卦，变动得是在第一爻，根据规则不变为体变为用

于是可得：【体卦】为上卦，震卦，五行属木，【用卦】为下卦，艮卦，五行属土。

木克土，体克用，可寻

梅花占失物（下） #

体用关系除开基本的生克，还要考虑时间和地点。

  1. 时间，即季节
    1. 每个季节都有五行的衰旺，比如夏天占卜的，夏天属火
    2. 体卦是离卦，则我很旺盛，因为离卦也属火，火在夏季是旺盛的
    3. 如果是冬天占卜的，体卦也是离卦，那我就不旺了，因为冬天属水，水会克主卦离卦
    4. 因此，在什么季节占卜，会导致体卦的衰旺，体卦太衰的话，克人家未必克的过，吉的力量也会减弱很多。、
    5. 反之，如果体卦很旺，也不怕别人来克我，伤害也会减少很多
  2. 地点
    1. 人在南极作占卜，则我的体卦也是离卦，那么我就不旺了，因为南极冰天雪地，五行属水，水会克我主卦

万物类象，这个概念出自《梅花易数》，它是将日常万物，按八卦的属性进行分门别类

宇宙万事万物都分别归类于八卦，因此一个卦所拟象的物类难以数计

射覆 #

这个提示通常是给时间，或者三个随机数字

其实，这就是古人玩的【射覆】

射覆，一种民间占卜猜物的游戏。在瓯、盂、碗等器具下覆盖某一物件，让人猜测里面是什么东西

《汉书·东方朔传》："上尝使诸数家射覆。"

颜师古注曰："于覆器之下而置诸物，令闇射之，故云射覆

射——猜度；覆——覆盖

（题外话，东方朔是汉武帝时期一个传奇人物，诙谐敏捷且善卜。这也是个常出现在影视剧中的人物形象，比如大汉天子、汉武帝等）

《西游记》里面也有这个情节

在某一集中，孙悟空跟国师斗法，猜柜子里的东西，就是这个玩法

《红楼梦》也有这个情节

在贾宝玉的生日宴上，他们玩游戏，第一次抓到的就是“射覆”

薛宝钗还说过这样一段话，“射覆从古有的，如今失了传，这是后人纂的，比一切令都难。这里头倒有一半是不会的，不如毁了，另挑一个雅俗共赏的”

（她的意思是，射覆很难，是最难的行酒令）

占财 #

eg:小 A 在做生意，预测财运，用报数起卦法：33 43

得到梅花卦：

本卦：天火同人（上乾下离） 互卦：天风垢 （上乾下巽） 变卦：风火家人（上巽下离）

步骤一：看卦词

本卦：是事情的开始，天火同人，中上卦，就是说要诚信经营，齐心合力，开局还不错哦

变卦：是事情的结果，风火家人，下下卦，不要看到下下卦就觉得一定很差，卦辞无绝对的吉凶，主要看是否贴切

上中下卦，都是后人的理解，原本的卦辞是无区分的，甚至没有绝对的凶吉，且不同的软件，给出的吉凶也不同，所以风火家人是什么情况呢？家人/团队给支持，会旺；反之呢，后院起火

步骤二：看体用关系：

《梅花易数》原文表达，占求财，以体为主，以用为财。体克用，有财；用克体，无财。体生用，财有损耗之忧；用生体，财有进益之喜。体用比和，财利快意，欲知得财之日，生体之卦气定之。欲知破财之日，克体卦气定之。

 又若卦中有体克用之卦，及生体之卦，则有财，此卦气即见财之日。若卦中有克体之卦，即体生用之卦，即破财，此卦气即破财之日。

那么上面的卦：体卦为离卦，五行火离火，用卦乾卦，五行金乾金，变卦巽卦，五行木，巽木

互卦表示事情的发展，分为上互卦，下互卦，此例中，上互卦为乾金，下互卦为巽木。

接下来，我们探讨下他们彼此的体用生克关系

体卦离火，用卦乾金

火克金（体克用），吉，有财

体卦离火，下互卦巽木

木生火，吉

体卦离火，上互卦乾金

火克金，吉

体卦离火，变卦巽木

木生火，吉

tip: 用卦是初始，互卦是过程，变卦是结果

最后结果：生进大吉卦，财有进益之喜。财运是好的

步骤三：回头看卦辞

既然这个卦求财为吉，为何卦辞还别有深意。是否还有其他提示给我们呢？

仔细看互卦、变卦

互卦，天风垢，有小人的意味（题外话：如果看感情，天风垢有“女小人”的含义，感情中出现第三方的意思）

变卦，风火家人，有家人/内部团队的含义。

综合以上三点，再来看应期

我们看卦，知道有财运之后，肯定还会好奇什么时候可以来财，财从哪个方向来呢？

看原文表达，“欲知得财之日，生体之卦气定之。欲知破财之日，克体气卦气定之”

在这个例子中，体卦为离火，并无克体卦气

只有两个生体卦气，这两个“生体卦”分别是：下互卦，巽木；变卦，巽母（体卦离火，木生火）

那么，得财之时就在“巽木”卦气

巽母=木气=3.4 月份=东南方（这是通过万物类象得来得）

总结： 小 A 做生意财运旺，尤其时农历三四月份财运好，且东南方向旺，但需要注意一下团队内部/合伙人之间的关系，避免发生在外赚了钱，在内被背刺得情况。
